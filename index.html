The genesis of this project is actually kind of silly. I had been going along book marking reading materials from my course for quite some time when I discovered that my browser hadn't saved a single one. So I decided to make a resource for myself and others. It's essentially a reading list for beginning data analyst enthusiasts taken almost exclusively from the Google Data Analytics Certificate Course. Not all links in the course have been included. Links to templates , learning logs, templates, etc. have been ignored since the object of this is not to recreate (especially since that would be plagiarism) the work of Google. However, they did provide a very robust list of extracurricular readings to check out. I have included what I thought was pertinent to include that.
